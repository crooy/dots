# e-Strategy dots

![image](https://i.cloudup.com/RCpB-ASfme.png)

WIP bootstrapping library for osx

## Installation

One-liner:

```
(mkdir -p /tmp/dots && cd /tmp/dots && curl -L https://github.com/paulrose/dots/archive/master.tar.gz | tar zx --strip 1 && sh ./install.sh)
```
